# DevKB-Mobile - Mobile App for Developer Knowledge Base

A mobile application for accessing your code snippets on the go.

## Features

- Cross-platform (iOS & Android)
- Offline access to snippets
- Quick search
- Dark mode
- Code syntax highlighting

## Tech Stack

- **Framework**: React Native / Expo
- **State**: Zustand
- **API**: DevKB-API

## Development

```bash
npm install
npm start
```
